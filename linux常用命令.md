## 文件与目录操作 

- / ：根目录
- ~：家目录
- ..：上一级目录

| 命令    | 作用          | 示例                  |
| ------- | ------------- | --------------------- |
| `ls`    | 列出目录内容  | `ls -l` 显示详细信息  |
| `cd`    | 切换目录      | `cd /home/user`       |
| `pwd`   | 显示当前路径  | `pwd`                 |
| `mkdir` | 创建目录      | `mkdir test`          |
| `rmdir` | 删除空目录    | `rmdir olddir`        |
| `rm`    | 删除文件/目录 | `rm -rf folder` ⚠危险 |
| `cp`    | 复制文件/目录 | `cp file1 file2`      |
| `mv`    | 移动或重命名  | `mv a.txt b.txt`      |
| `touch` | 新建空文件    | `touch new.txt`       |

### 文件压缩 & 解压 

| 命令                         | 作用     |
| ---------------------------- | -------- |
| `tar -czvf a.tar.gz folder/` | 压缩目录 |
| `tar -xzvf a.tar.gz`         | 解压     |
| `zip -r a.zip folder/`       | zip 压缩 |
| `unzip a.zip`                | zip 解压 |



## 文件查看 & 编辑 

| 命令      | 用途               | 示例                      |
| --------- | ------------------ | ------------------------- |
| `cat`     | 查看文本           | `cat file.txt`            |
| `tac`     | 反向查看           |                           |
| `head`    | 查看前 n 行        | `head -20 file.txt`       |
| `tail`    | 查看后 n 行        | `tail -50 file.txt`       |
| `tail -f` | 实时查看日志       | `tail -f /var/log/syslog` |
| `less`    | 分页查看           | `less file.txt`           |
| vim       | 高级编辑器（常用） | i 插入，：wq 保存并退出   |



## 查找 与 搜索 

### 在目录中搜索文件 

| 命令                        | 示例                 |
| --------------------------- | -------------------- |
| `find . -name "*.py"`       | 找到当前目录所有 .py |
| `find / -type f -size +10M` | 找大于 10MB 文件     |

### 内容搜索 

| 命令                      | 示例               |
| ------------------------- | ------------------ |
| `grep keyword file.txt`   | 搜索包含关键字的行 |
| `grep -r "hello" ./`      | 递归搜索所有文件   |
| `grep -n "text" file.txt` | 显示行号           |
| `grep -i`                 | 忽略大小写         |

## 权限、用户 与 系统管理 

### 权限管理 

| 命令                    | 作用       |
| ----------------------- | ---------- |
| `chmod 755 file`        | 改权限     |
| `chown user:group file` | 修改所有者 |
| `chgrp group file`      | 修改用户组 |

### 用户管理 

| 命令               | 作用         |
| ------------------ | ------------ |
| `adduser username` | 添加用户     |
| `passwd username`  | 修改密码     |
| `su user`          | 切换用户     |
| `sudo cmd`         | 以管理员执行 |

### 系统监控 

| 命令         | 作用                |

## 小技巧

```bash
# 查看命令帮助
man command
command --help

# 历史命令
history

# 重复执行上一条
!!

# 查找历史命令
Ctrl + R

# 清屏
clear

# 后台运行
command &

# 把程序守护运行
nohup python app.py &

# 查看你的 IP
ip a
hostname -I

# 快速返回上一个目录
cd -

# 查看文件大小（人类可读）
ls -lh

# 管道符（把前一个命令输出给后一个）
|

# 重定向(覆盖/追加)
> / >>

# 后台任务（继续运行/调回前台）
bg / fg
```



## 快捷键

### Linux 终端

#### 光标移动 

| 快捷键       | 功能           |
| ------------ | -------------- |
| **Ctrl + A** | 移到行首       |
| **Ctrl + E** | 移到行尾       |
| **Alt + B**  | 往回跳一个单词 |
| **Alt + F**  | 往前跳一个单词 |
| **Ctrl + B** | 左移一个字符   |
| **Ctrl + F** | 右移一个字符   |

#### 删除与编辑 

| 快捷键       | 功能                               |
| ------------ | ---------------------------------- |
| **Ctrl + U** | 删除光标前所有内容                 |
| **Ctrl + K** | 删除光标后所有内容                 |
| **Ctrl + W** | 删除前一个单词                     |
| **Ctrl + D** | 删除当前字符                       |
| **Ctrl + H** | 删除前一个字符（相当于 Backspace） |

#### 命令控制 

| 快捷键       | 功能                    |
| ------------ | ----------------------- |
| **Ctrl + C** | 终止当前命令            |
| **Ctrl + Z** | 暂停当前进程（放后台）  |
| **fg**       | 恢复前台运行            |
| **bg**       | 后台运行（结合 Ctrl+Z） |
| **Ctrl + L** | 清屏（等同 `clear`）    |
| **Ctrl + S** | 暂停屏幕输出            |
| **Ctrl + Q** | 恢复屏幕输出            |

#### 历史命令 

| 快捷键       | 功能                      |
| ------------ | ------------------------- |
| **Ctrl + R** | 搜索历史命令（超好用）    |
| **↑ / ↓**    | 浏览历史记录              |
| **!!**       | 执行上一条命令            |
| **!abc**     | 执行最近以 abc 开头的命令 |
