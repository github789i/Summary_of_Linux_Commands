# 大数据环境常用命令速查 



## 1. Hadoop 启动与停止

### **① 启动 Hadoop**

```bash
start-all.sh
```

### **② 停止 Hadoop**

```
stop-all.sh
```

------

##  2. HDFS 安全模式（Safe Mode）

### **强制退出安全模式**

```
hadoop dfsadmin -safemode forceExit
```

### **常用另一版本命令**

```
hdfs dfsadmin -safemode leave
```

------

##  3. MySQL

### **启动 MySQL**

```
mysql -uroot -proot
```

### **MySQL 配置常见参数（如 SSL）**

关闭 SSL 警告：

```
useSSL=False
```

------

##  4. Hive 启动

### **启动 HiveServer2**

```
hive --service hiveserver2 &
```

### **进入 beeline**

```
beeline
```

### **连接 Hive**

```
!connect jdbc:hive2://localhost:10000/default root root
```

------

##  5. Spark 启动

### **启动 Spark Master**

```
start-master.sh
```

### **运行 Spark SQL**

```
spark-sql
```
